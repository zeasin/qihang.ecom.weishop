# 启航视频号电商系统

## 一、项目概述

启航视频号电商系统是一套为专为微信视频号小店开发的一套电商业务系统，支持多店铺管理，主要功能：采购管理、店铺订单同步、发货管理、售后管理、商品管理、仓库出入库管理等，基本覆盖视频号小店电商业务处理完整流程，本系统后端采用Dubbo微服务架构，前端采用Vue2+Element开发。


## 二、技术框架
### 2.1 主体框架
+ Java 17
+ SpringBoot3.x
+ Dubbo3.x
+ Seata

### 2.2 中间件
+ Nacos2.2.0
+ Redis7.0.4
+ MySql8.x
+ Minio

### 2.3 前端
+ vue2
+ element
